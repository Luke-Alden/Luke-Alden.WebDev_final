<picture>

<source media = "(min-width: 800px)" <-The source element creates media condions. Media creates screen conditions for the image to render. srcset = "images/cats/kitten-large-650.png">

Embedded Media

For embedded media, we will look at how to embedd audio and video into your website. Lets start with the audio element.

<audio controls <-controls gives the user basic audio controls. >

<source src = "audio /alligator.mp3" <-This links to the audio file to display. type = "audio/mp3" <- This tells the website the type of audio to use.>

Sorry, your browser does not support this audio type. <- This is for the event that the user cannot use the audio file.
