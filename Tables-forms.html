Tables

To start out add this CSS to the head element: <style>td, th{border: 1px solid gray;}</style>. You'll learn how this works later but it will make your tables easier to read.

The <table></table> element sets up a table of rows and columns. A basic table is set up like this:

<table>

<caption>Title</caption> <- The caption element is where the table title is placed.

<thead> <- This is where the top row of the table goes.

<tr> <- This creates one row of the table.

<th> Row1,Column1</th> <- The th element crates one column and bolds the content.

<th>Row1,Column2</th>

</tr>

</thead>

<tbody> <- This is where the body of the table goes.

<tr>

<td>Row2,Column1</td> <- The td element also creates a column.

<td>Row2,Column2</td>

</tr>

</tbody>

</table>

Row1,Column1 Row1,Column2
Row2,Column1 Row2,Column2

<table>

Two attributes that are used with the <table> element are rowspan and colspan. The rowspan and colspan attributes control the span of the row and column respectively. This example:

<table>

<caption>Title</caption>

<thead>

<tr>

<th> Row1,Column1</th>

<th colspan = "2">Row1,Column2,3 </th>

</tr>

</thead>

<tbody>

<tr>

<td rowspan = "2">Row2,3,Column1 </td>

<td>Row2,Column2</td>

<td>Row2,Column3</td>

</tr>

<tr>

<td>Row3,Column1</td>

<td>Row3,Column2</td>

<td>Row2,Column3</td>

</tr>

</tbody>

</table>

Forms

There are many occasions in a website where you will need user input. To create a baisc form, will use the form element. Below is a sample of what a basic form looks like in code. Notice that I have added in a fieldset element to improve the look. I have also added div,br and a legend element to further improve the render.

<form action = "/api/user" <- This attribute declare where the info will be sent. method = "POST">

<fieldset> <- This creates a "box" around the contained code.

<legend>Insert Data</legend> <- This is where the title for fieldset is placed.

<div> <- This creates a line break at the end of the content.

<label>First Name:</label> <- This is where the label for the corresponding input element is.

<br>

<input type = "text" <- This declares the type of input. name = "first_name" id = "first_name">

</div>

<div>

<label>Last Name:

<br>

<input type = "text" name = "last_name" id = "last_name">

</div>

<div>

<label>Email:</label>

<br>

<input type = "email" <- The email value demands the user enter a valid email. name = "email" id = "emailfield">

</div>

<div>

<label>Password:</label>

<br>

<input type = "password" name = "password" id = "password" minlength = "7"> <- This attribute (along with maxlength) sets restrictions on the input.

</div>

<div>

<label>Bio:</label>

<br>

<textarea <- This element makes a text box, the size is declared with the rows and cols attribute. id = "biofield" name = "bio" rows = "4" cols = "25"></textarea>

</div>

<div>

<input type = "submit" value = "submit">

</div>

</fieldset>

</form>
