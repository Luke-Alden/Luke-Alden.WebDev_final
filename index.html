Setup

The core of HTML documents are elements and Attributes. Element and attribute values are setup as followed:

<elementName <-This is the opening tag attribute1 = attributeValue>content</elementName <- This is the closing tag>

Note that some elements like img do use a closing tag.

To start a basic HTML 5 document, you must write the follwoing lines of code:

<!DOCTYPE html>

<html> <- This element is the root element, it is the start of the code.

<head> <- This element is where code that controls the display is stored.

<meta charset=”utf-8”> <- This controls the encoding.

<title></title> <- This sets the title.

</head>

<body> <- This is where the stuff that will be displayed on screen is placed.

</body>

</html>

Notice that each nested element is indented with two spaces of each nest level.

Lists

There are three basic lists in HTML, unordered lists, ordered lists, and description lists. Unordered lists are built and appear like this:

Code

<ul>

<li>content-A</li>

<li>content-B</li>

</ul>

Appearance

  • content-A
  • content-B

Ordered lists are built and appear like this:

Code

<ol>

<li>content-A</li>

<li>content-B</li>

</ol>

Appearance

  1. content-A
  2. content-B

Description lists are composed of names and values, they are built and appear like this:

Code

<dl>

<dt>content-A</dt>

<dd>content-B</dd>

</dl>

Appearance

content-A
content-B
